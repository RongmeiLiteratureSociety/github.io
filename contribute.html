🤝 How to Contribute

We welcome participation from community members, linguists, developers, and anyone passionate about preserving the Rongmei language.

🔊 Submit Audio

Record native Rongmei words or sentences and email them to us at rongmeiliteraturesociety@gmail.com. Make sure to include the word meaning and speaker details (optional).

📝 Add Dictionary Entries

If you have new Rongmei words, idioms, or alternate meanings, you can submit them via GitHub pull request or by email in a plain text/CSV format to rongmeiliteraturesociety@gmail.com.

🛠️ Technical Help

Developers can help with audio upload tools, search interface, or auto-phonetic rendering tools. Open a GitHub issue or PR to get started.

Every small contribution helps build a stronger record of our language. 💚
